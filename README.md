Overview
This project is a Python-based Contact Management System that enables users to store and manage essential contact information such as Name, Phone Number, and Email Address.
The application uses CSV and JSON files for persistent data storage and supports structured error logging through exception handling to ensure reliability.

Objectives-
Learn and implement file handling using CSV and JSON in Python.
Perform CRUD operations (Create, Read, Update, Delete) on contact records.
Understand and apply exception handling and structured error logging.
Practice modular and structured programming in a console-based application.
Build a functional Python project demonstrating real-world problem-solving skills.

Files Used-
File Name	Purpose
contact_book.py -	Main program containing all functions and menu-driven logic.
contacts.csv -	Stores contact records (Name, Phone, Email) in tabular format.
contacts.json	- Contains exported contact data in JSON format and is used for import operations.
